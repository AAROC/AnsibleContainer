# AnsibleContainer

A repo for Ansible-ready containers [automated build](https://hub.docker.com/r/aaroc/ansiblecontainer) on Docker hub.

Two different base images are used :

  * CEntOS 6
  * CEntOS 7
  * Ubuntu 14.04
